# NAME
atomic-info - Display LABEL information about an image

# SYNOPSIS
**atomic info**
[**-h**]
IMAGE

# DESCRIPTION
**atomic info** displays the LABEL fields within an image

# OPTIONS:
**--help**
  Print usage statement
