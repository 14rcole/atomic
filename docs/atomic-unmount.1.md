# NAME
atomic-unmount - Unmount Images/Containers

# SYNOPSIS
**atomic unmount**
[**-h**|**--help**]
DIRECTORY

# DESCRIPTION
**atomic unmount** will unmount a container/image previously mounted with
**atomic mount**.

# OPTIONS:
**-h** **--help**
  Print usage statement
