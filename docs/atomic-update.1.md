# NAME
atomic-update - Pull latest Image from repository

# SYNOPSIS
**atomic update**
[**-h**]
IMAGE

# DESCRIPTION
**atomic update** will pull the latest update of the image from the repository

# OPTIONS:
**--help**
  Print usage statement
