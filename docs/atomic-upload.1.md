# NAME
atomic-upload - upload Image to repository

# SYNOPSIS
**atomic upload**
[**-p**][**--pulp**]
[**-h**]
IMAGE

# DESCRIPTION
**atomic upload** will upload of the image to the repository

# OPTIONS:
**-p** **--pulp**
  Upload using the pulp protocol, defaults to using docker push

**--help**
  Print usage statement
